# TinyGit

A simple local Git tool， based on CS61B(UC, Berkeley)

本项目是简易版的Git实现，支持本地仓库功能和本地仓库之间的remote相关命令，项目实现框架遵循CS61B中的proj2-gitlet，并通过了课程的gradescope中的所有测试
